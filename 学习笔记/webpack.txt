1、执行打包 <br>
安装webpack  npm install webpack@3.6.0 -g <br>
执行打包 D:\VUE\01-webpack的使用\01-webpack的起步> webpack .\src\main.js .\dist/bundle.js <br />
2、配置webpack.config.js <br>
3、初始化npm： npm init <br>
4、根据初始化的package.json的内容自动加载依赖  npm install <br>
5、webpack（全局webpack环境打包）打包或者 npm run build(本地webpack环境打包) 需要配置build <br>
6、安装本地项目的webpack版本， npm install webpack@3.6.0 --save-dev <br>
7、加载css-loader npm install --save-dev css-loader@2.0.2 但是css-loader只负责加载CSS，不会渲染CSS样式 <br>
8、加载style-loader   npm install --save-dev style-loader@0.23.1 负责将样式添加到dom中 <br>
9、加载less-loader npm install --save-dev less-loader@4.1.0 less@3.9.0 <br>
10、加载图片等url-loader npm install --save-dev url-loader@1.1.2
             npm install --save-dev file-loader@3.0.1 <br>
11、ES6语法处理成ES5 npm install --save-dev babel-loader@7 babel-core babel-preset-es2015 <br>
---------------------------------------------------------------- <br>
12、通过npm安装vue npm install vue --save <br>

13、解析vue后缀的文件 安装vueloader和vue编译 npm install vue-loader@15.4.2 vue-template-compiler@2.5.21 --save-dev <br >
   
14、打包html放到dist 安装html-webpack-plugin npm install html-webpack-plugin@3.2.0 --save-dev <br >

15、丑化js ： npm install uglifyjs-webpack-plugin@1.1.1 --save-dev<br >

16、自动build服务：npm install webpack-dev-server@2.9.3 --save-dev
